Project Hub

Project Hub is a web application that was designed to connect small businesses with developers for short term or long term projects. This was the capstone project for Codeup, a 16 week immersive full stack LAMP + J web development program. My team and I designed the page from concept to deployment using user stories and wireframes to setup the layouts. We utilized HTML, custom CSS, Javascript, Twitter Bootstrap and jQuery plugins for the front end design. Laravel's PHP MVC framework was utlized for mySQL database connection on the back end. We also implemented Laravel's built-in social authentication to allow developers to login or create a user using their GitHub credentials, and Amazon's S3 cloud service for file upload and storage. Project Hub can easily be customized to help any business find the right person for any project.
