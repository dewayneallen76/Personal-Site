Weather Map

The weather map utilizes the Open Weather Map API and Google Maps API. The basic design uses HTML, Bootstrap CSS, and custom CSS styling. Javascript and JQuery are used to make the map interactive. If you move the marker on the Google Map anywhere in the world, it will display the latitude and longitude, clicking Get Weather will change the three day forecast to the forecast for the location that the marker was placed.
